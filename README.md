# String Calculator Kata in Clojure

It's my first experience  with Clojure, I try to solve the [String Calculator Kata](https://github.com/xpeppers/string-calculator-kata)

## Run test

`lein test`

## Run from command line

`lein run 9,5`
`=> The sum is 14`
